# just-tech-news

## Description
An app where user posts an article from another source and other users vote or comment on the post.

## Technology
- REST API
- JawsDB Add-On (Heroku)
- Doteny
- bcrypt
- Sequelize (ORM)
- Node.js
- Express.js
- MySQL
- Handlebars
